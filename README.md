# Boilerplate for Metal JSX with Webpack

A starting point for developing a [Metal](https://metaljs.com/) app using JSX and Webpack. Also includes [Sass Loader](https://github.com/webpack-contrib/sass-loader).

Technologies:
- metal
- metal-jsx
- webpack
- babel
- sass-loader
